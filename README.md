# sophias-portfolio
## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **UI Library**: React 18
- **Styling**: Tailwind CSS (via utility classes)
- **Animations**: Framer Motion
- **Language**: TypeScript

## 📋 Pages

### Home

- Hero section with introduction
- Featured highlights of current initiatives
- Quick navigation to main sections

### Projects

- Categorized project display (Leadership, Ventures, Research)
- Detailed project descriptions with highlights
- Filterable project gallery

### Papers

- Research paper listings
- Expandable abstracts
- Keywords and paper metadata
- PDF download links

### Reports

- Strategic and market analysis reports
- Report categorization
- Key highlights and contents
- Download functionality

### About

- Personal biography
- Leadership and entrepreneurship sections
- Research focus areas
- Core values
- Contact information
